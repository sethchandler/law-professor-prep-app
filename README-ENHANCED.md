## ‚ú® What It Does

The Law Professor Assistant helps with:

### Teaching Preparation (Class Mode)
- Generate opening puzzles that put students on intelligent but wrong track
- Create one-fact-delta hypotheticals that locate doctrine precisely
- Develop counter-narrator scripts for defending losing parties
- Build complete class arcs with Socratic questioning

### Scholarly Analysis (Scholarship Mode)
- Transform intuitions into concrete claims with mechanisms
- Identify clean limits where ideas should lose
- Generate research vectors (specific sources to pursue)
- Develop strongest counter-arguments
- Design remedies and institutional reforms

### Creative Reframing (All Modes)
Apply ~12 "thinking moves" including:
- **Rename the problem** - Three frames, each implying different mechanisms
- **Remedy-first** - Start with better remedy, reverse-engineer rule
- **Anomaly hunt** - Find misfit cases that reveal real doctrine
- **Burden shift** - Move burden, predict litigation behavior
- **Counter-narrator** - Rewrite facts from losing party's view
